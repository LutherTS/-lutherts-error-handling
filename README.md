My personal library for how I handle errors across my projects under my own standards.

This project also serves as a showcase for how I develop my own libraries, maintain my file system organization, configure a project's Comment Variables, and make some of these Comment Variables publicly available.

For the ComVar Readonly demo, you can get the VS Code extension [here](https://comvar.lemonsqueezy.com/checkout/buy/f7f17eab-8a69-4cc8-a1cb-3a591b2be031). ...Or just run the `npm run comvar-readonly` command in the terminal.
